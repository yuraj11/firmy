#### Architecture highlights

- UI & Navigation: *Compose and Navigation compose, Material 3 design*
- Dependency injection: *Hilt*
- Storage: *Room database*
- Network: *Ktor, Kotlinx serialization*
- Logging: *Timber*
- Other: *Crashlytics*
